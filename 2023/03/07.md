# わかったこと

-   iTerm のインストールと以下ショートカットキー

```
    Option + space :　ターミナルの表示表示（Hotkey から設定できる）
    Command + t : 新しいタブの作成
    command + w : タブの削除
    command + 右左　 or 数字　:　タブの移動
    Command + shift + h : クリップボードの履歴の呼び出し
    Command + f : 検索
    Command + +か- : 拡大縮小
    Command + enter : フルスクリーンと解除
```
